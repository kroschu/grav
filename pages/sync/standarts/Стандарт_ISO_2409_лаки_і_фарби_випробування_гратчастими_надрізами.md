|--------------|-----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|------------|
| 0            | краї надрізів повністю гладкі; жоден з квадратів решітки не відшарувався.                                       |                                                                                                                                   |            |
| 1            | відшарування дрібних лусочок покриття на перетині надрізів; площа відшарувань трохи перевищує 5% площі решітки. | ![][1]                                                                                                                            |            |
